**MioNova** is a Windows desktop app for creators who want to turn TikTok Live interaction into real-time game events.

Viewer gifts, likes, follows, and comments can trigger in-game actions, overlay animations, sound alerts, text-to-speech, and automation hooks without forcing the streamer to leave the game.

MioNova currently focuses on **GTA V Single Player / Story Mode**, with the platform designed to support more games over time.

## What MioNova Is For

MioNova is built for live streamers who want their audience to affect gameplay directly. Instead of reading chat manually or switching windows during a stream, creators can map TikTok events into actions such as spawning enemies, triggering boss events, playing sounds, showing overlay widgets, or sending webhooks to local game integrations.

Typical use cases include:

- TikTok gift-to-game interaction
- GTA V interactive live streams
- Stream overlays for goals, top gifters, chat, spinner, and fireworks
- Soundboard and text-to-speech alerts
- Local action hooks using sounds, keystrokes, and webhooks
- Mod installation and update workflows for supported game packages

## Main Features

- **TikTok Live Connector**: connect a TikTok Live session and process gifts, likes, follows, comments, and joins in real time.
- **Gift Mapper**: map exact TikTok gifts or coin tiers to game commands and action hooks.
- **Action Hooks**: trigger sounds, keystrokes, local webhooks, or overlays from live events.
- **Overlay Studio**: create browser-source widgets for TikTok Studio or OBS.
- **Soundboard & Text-to-Speech**: play alert sounds or read live events aloud.
- **Mod Manager**: install, update, configure, and verify supported GTA V mod packages.
- **Auto Updates**: desktop releases are delivered through the public release channel.

## Safety Notice

MioNova is intended for **Single Player / Story Mode only**. Do not use MioNova mods in GTA Online or official multiplayer sessions.

For Steam users playing modded GTA V Story Mode, disable BattlEye before launching the game with mods, and re-enable it before playing GTA Online.
